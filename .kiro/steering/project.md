# Project Context

## Required Reading
At the start of every session, read the following files in the project root before doing anything else:
1. `detalhamento_entregas.md` — defines deliverables and constraints for the project.
2. `todo.txt` — tracks pending tasks that still need to be completed.

## Output Language and File
All solutions, explanations, and deliverables must be written in **Portuguese (pt-BR)** and saved in `Entrega.md` at the project root.
